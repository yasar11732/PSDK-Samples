===============================================
DX Surface Example 1 - DXSurf1
================================================
Last Updated: OCT. 18,  2001              

DESCRIPTION
============
DXSurface Sample 1 (DXSurf1) demonstrates a DXSurface BitBlt to a primary surface.


BROWSER/PLATFORM COMPATIBILITY
===============================
DXSurf1.exe operates with Microsoft® Internet Explorer 4 and greater on the 
Microsoft Win32® platforms.


USAGE
=====
To build DXSurf1.exe the following items are required:

1. Platform Software Development Kit (PSDK) (the latest version).

2. Internet Explorer 6 Public Preview header and library files that 
contains the DXTransform header and library files. (If using Internet Explorer 
6 Public Preview)   (If using IE 6 Public Preview)

3. Internet Explorer® 5.5 header and library files that contains the 
DXTransform header and library files. (If using Internet Explorer 5.5)

4. Microsoft Visual C++® 6.0 and greater

To run DXSurf1.exe:

1) Open DXSurf1.exe 
2) Go to the File Menu and choose from the available JPEGs
