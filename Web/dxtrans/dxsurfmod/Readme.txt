=============================
DX Surface Modifier Example - DXSurfMod
=============================
Last Updated: OCT. 18,  2001              

DESCRIPTION
============
DXSurfMod displays the affects of various DX Surface Modifiers


BROWSER/PLATFORM COMPATIBILITY
===============================
SurfMod.exe and DXETool.exe operate with Microsoft® Internet Explorer 5.5 and 
greater on the Microsoft Win32® platforms.

USAGE
=====
To build DXSurfMod.exe the following items are required:
1) Platform Software Development Kit (PSDK) Win32 (Latest).
2) Internet Explorer 6 Public Preview Headers and Libraries including the 
   DXTransforms Headers and Libraries. (If using IE 6 Public Preview)
3) Internet Explorer 5.5 Headers and Libraries including the  DXTransforms 
   Headers and Libraries. (If using IE 5.5)
4) Microsoft Visual C++ 6.0 and greater.
