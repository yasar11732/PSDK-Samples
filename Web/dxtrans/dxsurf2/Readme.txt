=============================
DX Surface Example - DXSurf2
=============================
Last Updated: OCT. 18,  2001              

DESCRIPTION
============
DXSurf2 demonstrates alpha blending of one surface with another.

BROWSER/PLATFORM COMPATIBILITY
===============================
DXSurf2.exe operates with Microsoft® Internet Explorer 4 and greater 
on the Microsoft Win32® platforms.

USAGE
=====
To build DXSurf3.exe, the following is required. 

1) Platform Software Development Kit (PSDK) (the latest version). 

2) If using If using Internet Explorer 6 Public Preview, the Internet Explorer 6 Public Preview header and library files that contains the DirectX Transform header and library files. 

3) If using Internet Explorer 5.5, the Internet Explorer 5.5 header and library files that contains the DirectX Transform header and library files. 

4) Microsoft® Visual C++® 6.0 or later

To run the demo go to the Demo folder and ensure that DXSurf2.exe and the accompanying files 
listed below are in the same folder. Then click DXSurf2.exe. 

Tulipshol2.jpg
Parrot.png
