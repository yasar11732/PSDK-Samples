Sample Application - XML_Order.exe - Read me
--------------------------------------------

This applications uses the msxml.dll from IE5.  The purpose of 
this application was to use XML in a Visual Basic application.

The program uses the following XML files:

users.xml - keeps all user names and passwords for the program. 
(Required - application cannot create this file)

log.xml - keeps track of who logs into the application. 
(Not-Required - application can create this file)

customers.xml - Holds all customers 
(Not-Required - application can create this file)

products.xml - Holds all products available for ordering 
(Required - application cannot create this file)

orders.xml - Holds all customer orders 
(Not-Required - application can create this file)
