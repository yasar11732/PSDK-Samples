XML Tree Viewer

Chris Lovett
December 1st 1998

It is simple VB application that populates a Tree Control from a loaded XML Document. The user can then navigate the XML document by expanding and collapsing nodes of the tree.

Installation

You must have VB6.0 installed already and a version of IE5 newer than bld 1102.. It depends on some POST IE5 Beta2 DOM changes. Then copy xmltree.exe to your machine and launch it.

Sources

The full source code and a VB 6.0 project file is available from http://xmlweb/msxml/samples/xmltree
