ADO: Maximizing Performance

Sample Overview

ADO's flexible object model offers developers a lot of choices, some of which perform better than others. This sample walks developers through several code examples demonstrating the highest performing techniques for creating parameters, choosing cursor location, specifying command types and more.

Setup

This sample requires no setup other than opening and running the Visual Basic project.

Running the ADO Maximizing Performance Sample

The ADO: Maximizing Performance sample is a Visual Basic program. To run the sample use Visual Basic to open PerfDemo.vbp. PerfDemo.vbp is located in Samples\ADO\Performance

System Requirements

This application requires the following system components to run.
  • Microsoft Windows NT 4.0 or Windows 2000
  • Microsoft Visual Basic version 6.0 or newer
  • SQL Server 7.0 or newer
