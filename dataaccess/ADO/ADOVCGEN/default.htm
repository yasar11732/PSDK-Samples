VC Client (a.k.a ADOVCGEN)

Sample Overview

The ADOVCGEN application uses VC++ to demonstrate:
  • How to connect to a backend database using the data links dialog box
  • How to use ADO to extract/ manipulate data and display it using the ADO Data Grid

Setup

Follow the instructions below to setup this sample.
  • Compile the ADOVCGEN project to generate the ADOVCGEN.EXE.

Running the Sample

Follow the instructions below to run this sample. 1. Connect to proper back-end database
  • Press the 'Build' button, this will bring up the 'Data Link Properties' Dialog box
  • Select the provider required to connect to the back-end database. Fill the appropriate entries in the Connection Tab for the Provider chosen (Refer to OLE DB Docs for details on the Data Link Properties Dialog Box )
  • Press the 'OK' Button, This will fill the 'Select Table' List Control and the 'Connect String' Edit Box.
2. Data Retrieval
  • Select a table from the 'Select Table' List Control by clicking on it.
  • You should be able to view data in the 'Data Grid Display' data control.
3. Data Retrieval / Manipulation
  • Type in the SQL Statement in the 'SQL Statement' Edit Box and press the 'Xecute' button. The support for SQL Statements (SELECT, INSERT, UPDATE and DELETE) is dependent on the Native Provider. The ADOVCGen Tool will forward all the SQL Statements to the Provider for processing.
  • You should be able to view data in the 'Data Grid Display' data control.
4. Refreshing the Query
  • Press the 'Refresh' Button.

System Requirements

This application requires the following system components to run.
  • Microsoft Windows NT 4.0 or Windows 2000
  • Microsoft Internet Information Services version 4.0 or 5.0
  • Microsoft Visual C++ version 6.0 or newer
  • SQL Server 7.0 or newer
