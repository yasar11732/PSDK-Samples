General Notes

In order to run these samples, your system must be running the Microsoft Windows XP Tablet PC Edition operating system or else have the Tablet PC Platform runtime redistributable installed. The redistributable is installed with the Microsoft Tablet PC Platform SDK.

You need to invoke the "Register the PSDK Directories with Visual Studio" Start Menu item in order to build several of the samples.

In order to build the Visual Basic 6 samples, the Tablet PC Platform runtime must be installed.

If you plan to use the managed code components of the Tablet PC Platform, install the Microsoft .NET Framework prior to installing the Tablet PC Platform SDK.

The .NET Framework is required to run the C# and Microsoft Visual Basic .NET samples. (The .NET Framework is automatically installed as part of Windows XP Tablet PC Edition.)

Either Visual Studio 6.0 with SP5 or Visual Studio .NET is required to compile the source code (managed code samples require Visual Studio .NET) of the Tablet PC Platform SDK samples.

When working with the Tablet PC Platform SDK samples, upgrading the Visual Basic 6.0 projects to Visual Basic .NET is not supported.

If you modify the code in the Tablet PC Platform SDK samples and later uninstall the SDK, any changes that you have made to the source code may be lost. Save the projects you modify in a folder that is not part of the platform software installation.
